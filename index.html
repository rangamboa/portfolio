About Me

Hello! I'm Ran, a web developer. I build websites.

The best websites are nice to look at, easy to navigate, simple to read and possibly fun to use! When a user returns to the website to accomplish the task at hand, I've done my job.

I create applications using a variety of methodologies and languages. Picking up a new way to build isn't a problem for me. I enjoy developing both front- and back-end software applications.

Please take a look at my projects below. They range from games to more serious business applications used by both local and international firms! If you'd be interested in working with me, please drop me a line below. Thanks for looking!
